# API Documentation

This folder contains comprehensive API documentation for the Oriana Order Tracking system.

## Available Documentation

### Category API
- **File:** [CategoryAPI.md](./CategoryAPI.md)
- **Base Path:** `/api/category`
- **Endpoints:** Create, Get All, Get by ID, Update, Delete

### Product API
- **File:** [ProductAPI.md](./ProductAPI.md)
- **Base Path:** `/api/product`
- **Endpoints:** Create, Get All, Get by ID, Update, Delete

### OEM API
- **File:** [OEMAPI.md](./OEMAPI.md)
- **Base Path:** `/api/oem`
- **Endpoints:** Create, Get All, Get by ID, Update, Delete

## Documentation Structure

Each API documentation file includes:
- Complete endpoint descriptions
- Request/response schemas
- Sample requests with JSON examples
- Sample responses (success and error cases)
- Path parameters and query parameters
- Request body structures
- Common error responses
- Important notes and constraints

## Usage

Refer to the individual API documentation files for detailed information about each endpoint, including:
- HTTP methods and paths
- Required and optional parameters
- Request body formats
- Response formats
- Error handling

## Base URL

All API endpoints are relative to the base API URL. The actual base URL depends on your deployment configuration.
